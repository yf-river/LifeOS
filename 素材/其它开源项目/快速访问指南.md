## 关键信息速查

### 推荐技术栈

| 层 | 选择 | 备选 | 学习资源 |
|----|------|------|--------|
| 后端 | Go | Python | memos 源码 |
| 前端 | React 18 | Vue 3 | blinko 源码 |
| 桌面 | Tauri | Electron | blinko 源码 |
| DB (本地) | SQLite | - | 代码参考 |
| DB (服务端) | PostgreSQL | MongoDB | WeKnora 参考 |
| 向量 DB | PG Vector | Milvus | khoj 参考 |
| API | gRPC-Gateway | tRPC | memos/blinko 源码 |

### 核心决策

1. **为什么 Go?** 
   - 编译快、部署简单、并发性能好
   - memos, WeKnora 都用 Go

2. **为什么 Tauri?**
   - 轻量级、性能好、开发快
   - blinko, Lumina-Note 都用 Tauri

3. **为什么 PostgreSQL?**
   - 生产级别、功能完整、PG Vector 支持
   - 主流选择

4. **为什么 MCP?**
   - 标准化工具协议
   - 与 Claude, Continue 等集成
   - blinko 深度集成
